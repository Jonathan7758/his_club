## 3. 验收标准（每阶段）

### MVP 验收标准
- [ ] 日扫描器可正常产出每日Top10
- [ ] 深度分析可产出 ≥ 3个差异化角度
- [ ] 公众号图文原料可直接导入现有管道
- [ ] 成本控制在 ¥1,500/月以内
- [ ] 系统稳定运行7天无故障

### V1.0 验收标准
- [ ] "历史如果"推演可产出 ≥ 3种走向
- [ ] 视频号脚本格式可直接使用
- [ ] 事实校验层可检测 > 80%的已知错误
- [ ] 完读率A/B测试启动
- [ ] 热/温/冷数据分级存储

### V2.0 验收标准
- [ ] 自动发布管道成功对接公众号API
- [ ] A/B测试框架可对比不同角度的阅读量
- [ ] 全自动工作流可在无人干预下完成选题→发布
- [ ] 月全量成本控制在 ¥3,000以内（标准运营）
